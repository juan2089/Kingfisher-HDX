This app was design and programmed by members of the Gross lab at Washington University in St. Louis.

o   readxl v1.4.2

o   openxlsx v4.2.5.2

o   writexl v1.4.2

o   rlist v0.4.6.2

o   seqinr v4.2.23

o   gridExtra v2.3

o   grid v4.2.2

o   devtools v2.4.5

o   shiny v1.7.4

o   shinydashboard v0.7.2

o   NGLVieweR v1.3.1

o   shinydisconnect v0.1.0

o   shinyBS v0.61.1

Kingfisher HDX-MS is an open-source code app. We encourage all users to get familiar with the code and modify it for their specific needs. Check the code here!
